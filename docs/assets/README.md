# Imágenes públicas de Adictos al Techno

Archivo local de las fotografías públicas disponibles en
`https://adictosaltechno.com/` al 18 de julio de 2026.

## Contenido

- 108 imágenes JPG únicas.
- 132 referencias editoriales inventariadas.
- 22 noticias, 22 eventos y 7 entrevistas revisadas.
- Imágenes principales, galerías e imágenes insertadas en el contenido.
- 0 errores de descarga.

Los archivos WebP, PNG y JPEG originales se normalizaron a JPG con calidad 92,
orientación EXIF corregida y fondo `#080A0D` cuando existía transparencia.

## Manifiestos

- `MANIFEST.csv`: relación fácil de abrir en una hoja de cálculo.
- `MANIFEST.json`: inventario completo, dimensiones y URLs de origen.

Una misma imagen puede aparecer en varias publicaciones. Los archivos se
deduplicaron por contenido y el manifiesto conserva todas sus referencias.

No se incluyeron logos, favicons, iconos de interfaz ni recursos de terceros que
no estuvieran servidos como fotografías desde el sitio o su API pública.
